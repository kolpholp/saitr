# Квадро — многопейджевый сайт

Содержимое:
- `index.html` — главная страница
- `portfolio.html` — галерея работ с фильтрами
- `reviews.html` — отзывы и видео
- `contact.html` — контакты, форма заявки и калькулятор
- `server.js` — отправка заявок в Telegram
- `assets/style.css` — общий стиль
- `assets/app.js` — общий скрипт
- `assets/photos/` — фото объектов
- `assets/videos/` — два видео в облегчённом размере

Как запустить локально:
1. Установите Node.js
2. В папке сайта выполните:
   npm install
   npm start
3. Откройте http://localhost:3000
